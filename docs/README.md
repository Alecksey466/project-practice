## Вариативная часть практики

Цель: Сделать возможным интерактивное прохождение викторин по экологии человека без чей-либо помощи с неограниченным числом попыток ответа на вопросы викторины.

Ход работы: В Figma создан новый проект, создан фреймворк для главной страницы викторин и все необходимые фреймворки для прохождения всех викторин с возможностью неограниченного числа ответов на каждый вопрос викторины.

Итог: В Figma был создан проект, позволяющий выбрать викторину и пройти её, с возможностью неограниченного числа попыток ответа на каждый вопрос каждой викторины. Теперь любой желающий без чей-либо помощи может попытаться ответить на все вопросы викторин верно с той или иной попытки.
